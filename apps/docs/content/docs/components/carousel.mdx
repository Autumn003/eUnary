### Props

Use the following props to customize the carousel.

| Prop                | Type                     | Description                                        |
| :------------------ | :----------------------- | :------------------------------------------------- |
| <Callout>slides</Callout>          | <Callout>slideData[]</Callout>          | Array of slide objects containing title, image source, and optional navigation details. |
| <Callout>className</Callout>          | <Callout>string</Callout>          | CSS class names for styling the outer carousel container. |


##### SlideData Interface

| Prop                | Type                     | Description                                        |
| :------------------ | :----------------------- | :------------------------------------------------- |
| <Callout>title</Callout>          | <Callout>string</Callout>          | Title of the slide, displayed as overlay text on the main image. |
| <Callout>imageSrc</Callout>          | <Callout>string</Callout>          | Image source URL for the slide. |
| <Callout>navigationTitle</Callout>          | <Callout>string</Callout>          | Label for the navigation button shown on the slide overlay. |
| <Callout>naviationLink</Callout>          | <Callout>string</Callout>          | Link URL to navigate when the navigation button is clicked. |
