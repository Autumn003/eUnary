### Props

Use the following props to customize the card.

| Prop                | Type                     | Description                                        |
| :------------------ | :----------------------- | :------------------------------------------------- |
| <Callout>title</Callout>             | <Callout>string</Callout>                   | The title of the card.                             |
| <Callout>subtitle</Callout>          | <Callout>string</Callout>                   | The sub-title of the card.                         |
| <Callout>description</Callout>       | <Callout>string</Callout>                   | The description of your card item.                 |
| <Callout>backgroundImage</Callout>   | <Callout>string</Callout>                   | URL of your background image.                      |
|<Callout>contextImage</Callout>      | <Callout>string</Callout>                   | URL of your product image.                         |
| <Callout>className</Callout>         | <Callout>string</Callout>                   | The CSS class to be applied to the card.           |
