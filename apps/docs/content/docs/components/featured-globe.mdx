### Props

Use the following props to customize the featured globe.

| Prop                  | Type                                           | Description                                                                 |
| :-------------------- | :--------------------------------------------- | :-------------------------------------------------------------------------- |
|  <Callout>globeClassName</Callout>             | <Callout>string</Callout>                                         | CSS class applied to the globe container                                     |
|  <Callout>containerClassName</Callout>    | <Callout>string</Callout>                                         | CSS class applied to the main container wrapper                                    |
|  <Callout>textClassName</Callout>              | <Callout>string</Callout>                                | CSS class applied to the animated text element                                                       |
|  <Callout>text</Callout>               | <Callout>string</Callout>                                     | Text content displayed over the globe                                 |
|  <Callout>textSize</Callout>                 | <Callout>"xl" \| "lg" \| "md" \| "sm"</Callout>                            | Size of the text overlay                                   |
|  <Callout>globeSize</Callout>        | <Callout>number</Callout>         | Size of the globe in pixels                                |
|  <Callout>globeRotateDirection</Callout>            | <Callout>"left" \| "right"</Callout>                                         | Direction of the globe's automatic rotation                         |
