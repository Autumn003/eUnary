### Props

Use the following props to customize the Reveal Pane.

| Prop                | Type                     | Description                                        |
| :------------------ | :----------------------- | :------------------------------------------------- |
| <Callout>leftImgSrc</Callout>          | <Callout>string</Callout>          | URL of left image |
| <Callout>rightImgSrc</Callout>          | <Callout>string</Callout>          | URL of right image |
| <Callout>className</Callout>         | <Callout>string</Callout>         | CSS class to be applied to the container           |
| <Callout>leftImgClassName</Callout>         | <Callout>string</Callout>         | CSS class to be applied on the left image           |
| <Callout>rightImgClassName</Callout>         | <Callout>string</Callout>         | CSS class to be applied on the right image           |
| <Callout>autoplay</Callout>         | <Callout>boolean</Callout>         | To enable automatic sliding           |
| <Callout>duration</Callout>         | <Callout>number</Callout>         | Duration in seconds for one interval           |
| <Callout>loop</Callout>         | <Callout>boolean</Callout>         | Enable loop for autoplay           |
| <Callout>slideDirection</Callout>         | <Callout>"left" or "right"</Callout>         | Direction of slider to move           |
| <Callout>pauseOnHover</Callout>         | <Callout>boolean</Callout>         | Stop the slider while hover the component           |
