### Props

<br/>
<Callout>Alert</Callout>

| Prop                | Type                     | Description                                        |
| :------------------ | :----------------------- | :------------------------------------------------- |
| <Callout>variant</Callout>           | <Callout>default</Callout> <Callout>processing</Callout> <Callout>error</Callout> <Callout>success</Callout>| Choose the variant of the alert.|
| <Callout>delay</Callout>             | <Callout>Number</Callout>                   | Disappearing time of the alert                     |
| <Callout>className</Callout>         | <Callout>string</Callout>                   | The CSS class to be applied to the alert.          |
| <Callout>children</Callout>          | <Callout>React.ReactNode</Callout>          | The child elements to be rendered within the alert.|



<Callout>AlertTitle</Callout>

| Prop                | Type                     | Description                                        |
| :------------------ | :----------------------- | :------------------------------------------------- |

| <Callout>className</Callout>         | <Callout>string</Callout>                   | The CSS class to be applied to the alert title.    |

| <Callout>className</Callout>         | <Callout>string</Callout>                   | The CSS class to be applied to the alert description.|
