### Props

<br/>
<Callout>`Alert`</Callout>

| Prop                | Type                     | Description                                        |
| :------------------ | :----------------------- | :------------------------------------------------- |
| `variant`           | `default` `processing` `error` `success`| Choose the variant of the alert.|
| `delay`             | Number                   | Disappearing time of the alert                     |
| `className`         | string                   | The CSS class to be applied to the alert.          |
| `children`          | React.ReactNode          | The child elements to be rendered within the alert.|



<Callout>`AlertTitle`</Callout>

| Prop                | Type                     | Description                                        |
| :------------------ | :----------------------- | :------------------------------------------------- |
| `children`          | React.ReactNode          | The child elements to be rendered within the title.|
| `className`         | string                   | The CSS class to be applied to the alert title.    |


<Callout>`AlertDescription`</Callout>

| Prop                | Type                     | Description                                        |
| :------------------ | :----------------------- | :------------------------------------------------- |
| `children`          | React.ReactNode          | The child elements to be rendered within the description.|
| `className`         | string                   | The CSS class to be applied to the alert description.|
