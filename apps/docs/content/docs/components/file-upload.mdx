### Props

Use the following props to configure the file upload component.

| Prop                | Type                     | Description                                        |
| :------------------ | :----------------------- | :------------------------------------------------- |
| <Callout>onChange</Callout>          | <Callout>(files: File[]) => void</Callout>          | Callback function that is called when files are uploaded or dropped. |
| <Callout>multiple</Callout>         | <Callout>boolean</Callout>         | Whether multiple files can be selected and uploaded at once. Defaults to false. |
