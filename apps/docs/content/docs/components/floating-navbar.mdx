### Props

Use the following props to configure the floating navbar.

| Prop | Type | Description |
| :--- | :--- | :---------- |
| <Callout>leftPill</Callout> | <Callout>PillConfig</Callout> | Configuration for the left pill button, including content, label, and optional URL. |
| <Callout>rightPill</Callout> | <Callout>PillConfig</Callout> | Configuration for the right pill button, including content, label, and optional URL. |
| <Callout>navLinks</Callout> | <Callout>NavLink[]</Callout> | Array of navigation links to display in the center section of the navbar. |
| <Callout>defaultActive</Callout> | <Callout>string</Callout> | Label of the nav item that should be active by default. Defaults to 'Home'. |
| <Callout>onNavigate</Callout> | <Callout>(url: string, label: string) => void</Callout> | Optional callback function triggered when a nav item is clicked. Receives the URL and label. |
| <Callout>className</Callout> | <Callout>string</Callout> | Additional CSS classes to apply to the outer container div. |
| <Callout>pillsClassName</Callout> | <Callout>string</Callout> | Additional CSS classes to apply to both left and right pill buttons. |

##### NavLink Interface

| Prop | Type | Description |
| :--- | :--- | :---------- |
| <Callout>url</Callout> | <Callout>string</Callout> | The destination URL for the navigation link. |
| <Callout>icon</Callout> | <Callout>React.ElementType</Callout> | Icon component to display alongside the label (e.g., Lucide icons). |
| <Callout>label</Callout> | <Callout>string</Callout> | Text label for the navigation item. Also used for active state tracking. |

##### PillConfig Interface

| Prop | Type | Description |
| :--- | :--- | :---------- |
| <Callout>url</Callout> | <Callout>string</Callout> | Optional destination URL when the pill is clicked. If not provided, pill won't navigate. |
| <Callout>content</Callout> | <Callout>React.ReactNode</Callout> | Content to render inside the pill (typically an icon or small component). |
| <Callout>label</Callout> | <Callout>string</Callout> | Text label that appears on hover (visible on screens â‰¥640px for right pill). |
