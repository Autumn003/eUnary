### Props

Use the following props to configure the payment card 3d.


| Prop                | Type                     | Description                                        |
| :------------------ | :----------------------- | :------------------------------------------------- |
| <Callout>cardHolder</Callout>          | <Callout>string</Callout>          | Name of the card holder displayed at the bottom-left of the card front. |
| <Callout>cardNumber</Callout>          | <Callout>number</Callout>          | Full card number (16 digits). Automatically formatted and masked on the back side. |
| <Callout>cardExpiryMonth</Callout>         | <Callout>number</Callout>         | Expiry month displayed on the back side (between 1 to 12).           |
| <Callout>cardExpiryYear</Callout>         | <Callout>number</Callout>         | Expiry year of your card displayed on the back side           |
| <Callout>tascardBankNameks</Callout>         | <Callout>string</Callout>         | Bank name or issuer label shown on the front of the card.           |
| <Callout>themeColor</Callout>         | <Callout>string</Callout>         | Accent color (in HEX or CSS color) applied to highlights, shadows, and text glow.           |
| <Callout>chipColor</Callout>         | <Callout> "silver" \| "gold"</Callout>         | Chip color variant for the front card chip. Defaults to 'gold'.           |


##### CardFace Interface

| Prop                | Type                     | Description                                        |
| :------------------ | :----------------------- | :------------------------------------------------- |
| <Callout>isVisible</Callout>          | <Callout>boolean</Callout>          | Determines if this face is currently visible (used to control opacity and front/back visibility during flip). |
| <Callout>glareStyle</Callout>          | <Callout>React.CSSProperties</Callout>          | Inline CSS style for the animated glare effect that follows mouse movement. |
| <Callout>baseClass</Callout>          | <Callout>string</Callout>          | CSS classes defining the gradient background, shadows, and overall look of the card face. |
| <Callout>style</Callout>          | <Callout>React.CSSProperties</Callout>          | Additional inline styles for custom transformations or layout (e.g., rotating the back face) |
| <Callout>children</Callout>          | <Callout>React.ReactNode</Callout>          | Content to render inside the card face (e.g., chip, card number, CVV, etc.) |
