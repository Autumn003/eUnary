          return twMerge(clsx(inputs));

### Props

Use the following props to customize the fliping words.

| Prop                | Type                     | Description                                        |
| :------------------ | :----------------------- | :------------------------------------------------- |
| <Callout>words</Callout>          | <Callout>string[]</Callout>          | Array of words to be displayed. |
| <Callout>className</Callout>         | <Callout>string</Callout>         | The CSS class to be applied to the card.           |
| <Callout>duration</Callout>         | <Callout>number</Callout>         | Interval duration in milliseconds between word transitions.           |
