### Props

Use the following props to configure the progress card.


| Prop                | Type                     | Description                                        |
| :------------------ | :----------------------- | :------------------------------------------------- |
| <Callout>title</Callout>          | <Callout>string</Callout>          | Title displayed at the top of the card. |
| <Callout>description</Callout>          | <Callout>string</Callout>          | Short description or subtitle below the title. |
| <Callout>themeColor</Callout>         | <Callout>string</Callout>         | Custom theme color in hex code for progress ring, accents, and highlights.           |
| <Callout>percentage</Callout>         | <Callout>number</Callout>         | Overall completion percentage (0â€“100). If provided, circular progress is shown.           |
| <Callout>tasks</Callout>         | <Callout>task[]</Callout>         | List of tasks with icon, label, and completion data.           |
| <Callout>className</Callout>         | <Callout>string</Callout>         | Additional CSS classes to customize the card container styling.           |


##### Task Interface

| Prop                | Type                     | Description                                        |
| :------------------ | :----------------------- | :------------------------------------------------- |
| <Callout>icon</Callout>          | <Callout>ComponentType</Callout>          | Icon component displayed alongside the task label. |
| <Callout>label</Callout>          | <Callout>string</Callout>          | Label of the task. |
| <Callout>completed</Callout>          | <Callout>number</Callout>          | Number of completed items for this task. |
| <Callout>total</Callout>          | <Callout>number</Callout>          | Total number of items for this task. |
